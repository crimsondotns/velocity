🚀 คู่มือการตั้งค่า PM2 รัน Python (Dexscreener Bot) อัตโนมัติ 24/7 บน Windows

คู่มือฉบับนี้จะช่วยให้คุณสามารถรันบอท Python อยู่เบื้องหลัง (Background) แบบไม่มีหน้าต่างดำกวนใจ และเมื่อคอมพิวเตอร์รีสตาร์ทหรือไฟดับ บอทจะตื่นขึ้นมาทำงานเองอัตโนมัติทันที

🛠️ ส่วนที่ 1: การสั่งรันบอทแบบซ่อนหน้าต่าง (เริ่มต้นใช้งาน)

เพื่อให้บอทรันแบบไร้หน้าต่างกวนใจ เราจะใช้ไฟล์ runner.js เป็นตัวกลางในการเรียก Python แทน

เปิด PowerShell 2. เข้าไปที่โฟลเดอร์โปรเจกต์ของคุณ:

# เปลี่ยน Path ด้านล่างให้เป็นที่อยู่ที่เก็บไฟล์โปรเจกต์ในเครื่องของคุณจริงๆ
cd "C:\พาธ\ไปยัง\โฟลเดอร์\โปรเจกต์ของคุณ"


สั่งรันบอทผ่าน PM2:

pm2 start runner.js --name "dexscreener"


(หมายเหตุ: ตอนนี้บอททำงานแล้ว แต่ถ้ารีสตาร์ทเครื่อง บอทจะยังไม่เปิดตัวเอง)

⚙️ ส่วนที่ 2: ตั้งค่าเปิดคอมแล้วรันอัตโนมัติ (Startup)

ทำเพียง ครั้งเดียวจบตลอดชีพ เพื่อฝัง PM2 เข้ากับระบบ Startup ของ Windows

ติดตั้งแพ็กเกจเสริมสำหรับ Windows (ใช้ pnpm):

pnpm add -g pm2-windows-startup


สั่งให้ PM2 ฝังตัวเข้ากับระบบเปิดเครื่อง:

pm2-startup install


บันทึกสถานะบอทที่รันอยู่ปัจจุบัน (สำคัญมาก!):

pm2 save


💡 pm2 save คือการบอกให้ PM2 จำไว้ว่า "ตอนนี้มีบอทชื่อ dexscreener รันอยู่นะ รอบหน้าที่เปิดคอม ให้ไปเรียกตัวนี้กลับมาทำงานด้วย!"

🎉 เสร็จสิ้น! ต่อจากนี้ไป ไม่ว่าไฟจะดับ คอมจะค้าง หรือสั่ง Restart เครื่อง... ทันทีที่ Windows เปิดขึ้นมาใหม่ บอทจะแอบทำงานอยู่หลังบ้านให้ทันทีแบบเงียบเชียบ (ไม่มีหน้าต่างดำโผล่มาเลย)

🔍 ส่วนที่ 3: คำสั่งที่ใช้บ่อย (เช็คสถานะ & ดู Log)

เมื่อระบบรันอยู่เบื้องหลัง คุณสามารถเปิด PowerShell ขึ้นมาเพื่อเช็คการทำงานได้ตลอดเวลา:

ดู Log สดๆ (Real-time):

pm2 logs dexscreener


(กด Ctrl + C เพื่อออกจากหน้าต่าง Log โดยที่บอทยังทำงานอยู่)

ดูสถานะบอททั้งหมด (ว่ารันอยู่ไหม กิน RAM เท่าไหร่):

pm2 status


รีสตาร์ทบอทใหม่ (เช่น หลังจากแก้โค้ดเสร็จ):

pm2 restart dexscreener


🛑 ส่วนที่ 4: วิธียกเลิก / ปิดการทำงาน

หากวันหนึ่งคุณต้องการหยุด หรือยกเลิกระบบรันอัตโนมัติ ให้เลือกทำตามกรณีด้านล่างครับ:

กรณีที่ 1: แค่อยากหยุดชั่วคราว (เปิดคอมรอบหน้าก็จะไม่รัน จนกว่าจะสั่ง start เอง)

pm2 stop dexscreener


กรณีที่ 2: เลิกใช้บอทตัวนี้ถาวร (ลบออกจาก PM2)

pm2 delete dexscreener
pm2 save


(ถ้าไม่พิมพ์ pm2 save ตามหลัง เปิดคอมรอบหน้า PM2 อาจจะไปงัดมันกลับมาทำงานอีก)

กรณีที่ 3: ถอนรากถอนโคนระบบ Startup (ไม่ให้ PM2 ยุ่งกับการเปิดเครื่องอีกต่อไป)

pm2-startup uninstall
